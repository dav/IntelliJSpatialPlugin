# Spatial <version>

## Install

Download the attached plugin ZIP from this release and install it in a
compatible JetBrains IDE:

`Settings/Preferences > Plugins > gear icon > Install Plugin from Disk...`

## Notes

- Minimum compatible IDE build: `261` (`2026.1+`)
- This plugin depends on the bundled MCP Server plugin in supported IDEs.
